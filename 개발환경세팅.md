#### 웹개발기초 교과목 멘토링

**IDE란 Integrated Development Environment의 약자**로, 우리말로 번역하면 **통합 개발 환경**이다.

다양한 IDE중 서브라임 텍스트를 통해 web front-end를 배워봅시다.

**서브라임 공식 홈페이지**에서 다운로드가 가능하다. 설치가 완료되면 실행시켜주자!

본격적으로 코드를 작성하기 위해서 제일 먼저 해야 할 작업은 워크스페이스를 만들어 주는 것이다.

워크스페이스는 작업 공간이라고 불리는 폴더이며, 앞으로 모든 프로젝트가 저장될 최상단 폴더이다. 

여러 프로젝트는 워크스페이스로 분류하여 잘 관리해두자!

이제는 소스 파일을 생성해 보도록 할 것이다. 

생성한 하위 패키지를 오른쪽 클릭한 이후 New File을 눌러 새로운 파일을 생성해 주면 된다.

untitled라는 파일이 하나 생성되는데, 이름을 바꾸어 다른 이름으로 저장해주자!

지금은 html 확장자로 저장하여 html 파일을 생성해보도록 한다. 

**주의!**

**서브라임 텍스트는 소스 파일의 확장자를 통해 사용 언어를 구분하기 때문에 확장자는 항상 알맞게 입력해주기!!!!**

여기서 끝이 아니라 마지막으로 할 일이 하나 더 남았다.

**플러그인**이라는 것을 추가로 설치해주기!

위에서 플러그인에 대해 잠깐 언급하였는데, 플러그인이란 무엇일까요?

플러그인이란 **메인 프로그램에 추가되는 확장 기능**을 말한다. 

예를 들어 서브라임 텍스트의 기본 버전은 HTML 태그 작성시 여는 태그를 자동으로 닫아주는 기능이 없는데, 자동 태그닫기 플러그인을 설치함으로써 서브라임 텍스트에 태그를 자동으로 닫아주는 기능을 추가할 수 있다.
